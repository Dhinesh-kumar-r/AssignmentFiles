HTML TERMS

HTML
Hypertext Markup Language
HTML Element
An HTML element is defined by a start tag, some content, and an end tag.
HTML Tag
HTML tags are keywords that tell a browser how to format and display information on a page.
Angle Bracket
In HTML, angle brackets (< and >) are used to enclose the name of an HTML element
HTML Attribute
HTML attributes are pieces of markup language that can change the behavior or display of an HTML element. For example, attributes can change the color, size, or functionality of HTML elements.
HTML Comment
This is a comment. Comments are not displayed in the browser>
HTML Object Element
The HTML element represents an external resource that can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.
